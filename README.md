# Ability Beyond Disability

> **Ability Beyond Disability** (Arabic: **رغم إعاقتي أبدع**) is a responsive landing page designed to showcase and celebrate the talents, creativity, and achievements of people with disabilities in Algeria.

## 🌟 Overview

This project is a modern, accessible, and visually engaging static website built using pure HTML and CSS. It provides a platform where talented individuals with disabilities can present their work, participate in events, and share their stories with the community.

The website aims to:

- Highlight artistic and professional talents.
- Promote upcoming events and workshops.
- Showcase creative works in a gallery.
- Inspire others through success stories.
- Build a supportive and inclusive community.

---

## 🎯 Features

### 🏠 Hero Section
- Inspiring headline and mission statement.
- Call-to-action buttons:
  - Explore Talents
  - Showcase Your Talent
- "Talent of the Week" featured card.

### 📊 Statistics Section
Displays key numbers such as:
- Registered talents
- Represented provinces
- Published artworks
- National events

### 🎨 Talents Section
- Responsive grid of featured talents.
- Includes specialization, location, and tags.

### 📅 Events Section
- Upcoming workshops, exhibitions, and competitions.
- Featured national event with detailed information.

### 🖼️ Gallery Section
- Creative showcase of artworks.
- Interactive hover effects.

### 📬 Newsletter Subscription
- Email subscription form.

### 📌 Footer
- Project branding and copyright.
